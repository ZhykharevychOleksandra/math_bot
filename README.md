# Telegram Math Bot

A simple Telegram bot in Python (aiogram library).
The bot helps train math skills by generating random examples and working as a calculator.

## Main features

*  **Calculator**: Supports basic operations (+, -,*, /).
* **Math problems**: Generates random multiplication examples.
* **Interactivity**: Convenient control via Telegram commands.

## Technologies

* **Python 3.10+**
* **aiogram 3.x**
* **python-dotenv** (for secure token storage)

## Bot link
[@math_helper_dev_bot](https://t.me)

## How to run the project locally

1. **Clone the repository:**
```bash```
git clone https://github.com
cd math_bot

2. Install the dependencies:
pip install -r requirements.txt

3. Run the bot:
python main.py
